1.5 使用Node.js执行JS脚本文件

Hello World

  1. 创建编写JS脚本文件
  2. 打开终端，定位到脚本文件所属目录
  3. 输入 node 文件名 执行对应的文件

注意：文件不要使用 node.js 来命名文件名，执行的时候会打开这个文件，除了node.js这个名字，随便起，而且最好不要使用中文

  • 解析执行JavaScript
  • 读写文件
  • HTTP

helloworld.js

var foo = 'hello nodejs'
console.log(foo)

打开命令行工具，定位到你要执行的文件所属的目录

$node 00-helloworld.js 
hello nodejs
