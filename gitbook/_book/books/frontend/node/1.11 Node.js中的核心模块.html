1.11 Node.js中的核心模块

核心模块

Node.js为JavaScript提供了很多服务器级别的API，这些API绝大多数都被包装到了一个具名的核心模块中了。

  • 文件操作的fs核心模块
  • HTTP服务构建的http模块
  • path路径操作模块
  • os操作系统信息模块

只要这个模块是一个核心模块，你就马上要想到如果要使用它，就必须：

var fs = require('fs')
var http = require('http')
  • fs.writeFile
  • fs.unlink（删文件）
  • fs.mkdir
  • fs.readdir
  • http.createServer

var os = require('os')
