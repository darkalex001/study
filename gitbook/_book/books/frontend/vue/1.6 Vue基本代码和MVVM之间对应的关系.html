1.6 Vue基本代码和MVVM之间对应的关系

  1. 导入Vue的包
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

  1. 创建一个Vue的实例

    当我们导入包之后，在浏览的内存中，就多了一个Vue构造函数

el 表示，当前我们 new 的这个 Vue 实例，要控制页面上的哪个区域

data data 属性中，存放的是 el 中要用到的数据

msg 通过 Vue 提供的指令，很方便的就能把数据渲染到页面上，程序员不再手动操作DOM元素了【前端的Vue之类的框架，不提倡我们去手动操作DOM元素了】

var vm = new Vue({
    el:"#app"
    data:{
        msg: '欢迎学习Vue'
    }
})

将来 new 的Vue实例，会控制这个app元素中的所有内容，Vue 实例所控制的这个元素区域，就是我们的 V

<div id="app">
    <p>{{ msg }}</p>
</div>
